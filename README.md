
## Image Tagger

This little app loads images from an external API and then allows to add, edit and remove tags associated with the images.
It also saves your work as you do it. Feel free to refresh the page and see that you work wasn't lost :)


## Running Locally

First, clone this repository. Then cd into it.
Finally, run the following command:

npm run install && npm run dev

Alternatively, you can use Yarn:

yarn && yarn dev
